Política de Privacidade

Este site não coleta, armazena ou compartilha nenhum dado pessoal.

Não há cookies, rastreadores, análises (analytics) ou serviços de terceiros rodando neste site.

Os únicos dados armazenados localmente são as suas cédulas de bingo, imagens adicionadas, e o tema escolhido — tudo salvo no localStorage do seu navegador. Esses dados ficam apenas no seu dispositivo e nunca são enviados para nenhum servidor.

Este site é hospedado pelo GitHub Pages. Veja o aviso de privacidade do GitHub para informações sobre o que o GitHub pode coletar em nível de infraestrutura.

Privacy Policy

This website does not collect, store, or share any personal data.

There are no cookies, trackers, analytics, or third-party services running on this site.

The only data stored locally is your bingo cards, any images you've added, and your chosen theme — all saved in your browser's localStorage. This stays on your device and is never sent anywhere.

This site is hosted by GitHub Pages. See GitHub's privacy notice for information on what GitHub may collect at the infrastructure level.
